Docker 桌面版（Docker Desktop）是Docker退出的可以在桌面操作系统上管理容器化环境的工具，SmartIDE本地模式需要调用Docker桌面版以便管理容器环境。安装这个工具其实非常简单，但是安装之后最好对工具的一些默认设置进行一些调整一遍让SmartIDE运行得更加高效。

### 下载安装包

Docker桌面是一款免费工具，大家可以从以下两个渠道下载：

- 官网下载地址 https://www.docker.com
- SmartIDE所提供的国内镜像
  - [Intel芯片版下载地址](https://smartidedl.blob.core.chinacloudapi.cn/docker/20210926/Docker-osx-intel.dmg)
  - [Apple芯片版下载地址（M版）](https://smartidedl.blob.core.chinacloudapi.cn/docker/20210926/Docker-osx-apple.dmg)
